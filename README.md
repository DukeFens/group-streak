## 📜 Luật chơi

1. Mỗi ngày ít nhất **1 contribution** (commit, PR, issue, review) — repo nào cũng được.
2. Đứt streak → khao trà sữa cả nhóm 🧋
3. Bảng xếp hạng tính theo **current streak**, hoà thì xét **tổng contribution 365 ngày**.

## ➕ Thêm thành viên

1. Thêm username vào file [`members.txt`](./members.txt) (mỗi dòng một username)
2. Thêm một streak card ở mục trên (copy một block, đổi username)
3. Commit — leaderboard sẽ tự cập nhật ở lần chạy tiếp theo
